# Fake Profile Detector

## 📌 Project Description

Fake Profile Detector is a Machine Learning based web application that detects whether a social media profile is real or fake using trained data.
The model is trained using a dataset and integrated with a web interface built using Flask.

## 🚀 Features

* Detect fake social media profiles
* Machine learning model prediction
* Simple web interface
* Uses trained dataset
* Fast and easy to use

## 🛠️ Technologies Used

* Python
* Flask
* Machine Learning
* Pandas
* Scikit-learn
* HTML / CSS

## ▶️ How to Run the Project

1. Install required libraries

pip install flask pandas scikit-learn

2. Train the model (optional)

python train_model.py

3. Run the app

python app.py

4. Open in browser

http://127.0.0.1:5000/
